# Squid-Game-Dashboard-EXCEL-
Squid Game Dashboard (EXCEL)
•	This project presents an interactive analytical dashboard built on the Squid Game Season

•	This dashboard allows viewers to:
o	Explore the storyline data visually
o	Understand character progression
o	Analyze death patterns, survival stats, and game difficulty
o	Gain a clear summary of Season 1 events in a structured, interactive format
